## Content 

For blog 4, I chose <b>Animate.css</b> as my tool. Animate.css is a website that lets you try different basic animations. These include:

* Different exit animations (Rotating exits, fading exits, bouncing exits, etc)
* Different entrance animations (Back entrances, sliding entrances, zooming entrances, etc).



 I tinkered with Animate.css by clicking, previewing and seeing what all the animations on the website did. I just played around and tested the animations on the website itself. I also checked out some of the animations on an HTML tester webpage. I enjoyed the effects and the changed feel of the animations. This helped me get an idea of the different options I could choose from and what I can use in the future for an intended website if I decide on it or if im assigned work for it. 

#### Coding Section for Content:


```html
<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/animate.css/4.1.1/animate.min.css"/>

<style>
 
.tinker-fast {
  --animate-duration: 0.7s;
}

.tinker-slow {
  --animate-duration: 2.5s;
}

.tinker-delay {
  --animate-delay: 0.5s;
}

.tinker-hover:hover {
  animation: pulse 1s;
}
</style>
</head>

<body>

<h1>
Silly Blog Post
</h1>

<p>
button.. called the silly button. But it doesn't do anything. button lazy... i now call it couch potato
</p>
<!-- class="animate__animated animate__slideInLeft tinker-slow" -->
<button>
Silly Mode
</button>

<div class="animate__animated animate__slideInLeft tinker-slow">
*yayy*
</div>

</body>
</html>
```
